網戸の掃除のやり方【初心者でも簡単】

公開日: 2026-06-26

網戸掃除窓掃除ハウスクリーニング掃除術お掃除初心者

網戸の汚れ、気になっていませんか？

「なんとなく部屋が暗い」「窓を開けると砂っぽいにおいがする」そんなときは、網戸が汚れているサインかもしれません。網戸は目が細かいぶん、ホコリ・花粉・排気ガスの油汚れなどが絡みつきやすく、放置すると落としにくくなってしまいます。

でも安心してください。正しい道具と順番さえ覚えれば、初めてでも30〜60分できれいにできます。この記事では「外さずに掃除する方法」と「外してしっかり洗う方法」の両方をご紹介します。

掃除の前に準備するもの

  • 古い新聞紙またはダンボール（窓の内側に当てる用）
  • 掃除機（ブラシノズルがあると便利）
  • 柔らかいブラシまたは網戸専用ブラシ
  • マイクロファイバークロス2枚以上
  • 中性洗剤（食器用洗剤でOK）
  • バケツ・スポンジ
  • 軍手（ロールブラシ代わりに使えます）

特別な洗剤は不要です。家にある食器用の中性洗剤で十分汚れが落とせます。

【外さずにできる】網戸の掃除手順

網戸を外すのが面倒なとき、2階以上で外すのが不安なときは、この方法を試してみてください。

  1. 窓の内側に新聞紙をあてる：網戸の内側（室内側）にダンボールや新聞紙を貼り付けておくと、掃除のときに汚れが室内に入り込みません。
  2. 掃除機でホコリを吸う：ブラシノズルを使い、網戸の外側から軽くなぞるようにホコリを吸い取ります。力を入れすぎると網が歪むので、やさしく行いましょう。
  3. 濡らした軍手で拭く：軍手を両手にはめ、水で濡らして固く絞ります。網戸の外側から指で網目をつまむようにして拭くと、細かい汚れまで取れます。
  4. 乾いたクロスで仕上げ拭き：最後に乾いたマイクロファイバークロスで水分を拭き取れば完了です。

【外してしっかり洗う】網戸の掃除手順

年に1〜2回は網戸を外して丸洗いするのがおすすめです。油汚れや花粉がこびりついた頑固な汚れもすっきり落とせます。

  1. 網戸を取り外す：網戸の上部を室内側に少し押し込みながら手前に引くと外れます。メーカーや種類によって外し方が異なる場合があるので、取扱説明書も参考にしてください。
  2. 地面に養生シートを敷いて置く：ベランダや玄関先にレジャーシートや古いシートを敷き、その上に網戸を水平に置きます。
  3. 水で予洗いする：シャワーや水をかけて表面のホコリを大まかに流します。
  4. 洗剤液をスポンジで塗る：バケツにぬるま湯と食器用中性洗剤を溶かし、スポンジに含ませて両面をやさしくこすります。力を入れすぎず、円を描くようになでてください。
  5. 水でしっかりすすぐ：洗剤が残ると汚れが再付着しやすくなります。泡がなくなるまで丁寧に流しましょう。
  6. 日陰で乾かして取り付ける：直射日光に当てると網が縮む可能性があるため、風通しの良い日陰で乾かしてから元の位置に戻します。

掃除をもっとラクにするコツ

掃除のタイミングは「曇りの日」がベスト

晴れた日は洗剤がすぐに乾いて跡が残りやすいため、曇りや風のない日に行うのがおすすめです。

ロールブラシを活用しよう

100円ショップなどで売っている「網戸用ロールブラシ」を使うと、両面を同時に挟んで拭けるのでとても便利です。軍手の代わりに使ってみてください。

汚れ防止スプレーで次の掃除をラクに

掃除後に市販の「網戸用防汚スプレー」を吹き付けておくと、静電気によるホコリの付着を抑えられます。次の掃除がぐっとラクになります。

よくある失敗と注意点

失敗①：内側だけ拭いて汚れが広がった

網戸の汚れは外側に多いため、必ず外側から掃除しましょう。内側だけ拭くと汚れを室内側に押し込んでしまいます。

失敗②：強くこすって網が歪んだ

スポンジやブラシで強くこすりすぎると、網目が広がったり変形したりします。常にやさしい力を意識してください。

失敗③：洗剤をすすぎ忘れた

洗剤が残ると次のホコリがくっつきやすくなります。すすぎは「泡が完全になくなるまで」が鉄則です。

まとめ

網戸の掃除は、道具と手順さえ押さえれば初めてでも失敗しません。まずは「外さずに拭く方法」から試してみて、慣れてきたら年に1〜2回の「丸洗い」にも挑戦してみましょう。きれいな網戸は室内への空気の流れも改善し、毎日の換気が気持ちよくなりますよ。ぜひ今日から試してみてください。

よくある質問

網戸の掃除はどのくらいの頻度でやればいいですか？

外側の拭き掃除は月に1回程度、丸洗いは年に1〜2回（花粉シーズン後の5月と、台風シーズン後の10月ごろ）が目安です。汚れが目立ってきたら早めに対処するとラクに落とせます。

網戸を外さずにきれいにする方法はありますか？

あります。窓の内側に新聞紙をあてた上で、濡れ軍手や網戸専用ロールブラシを使って外側から拭くだけでも十分きれいになります。軽い汚れなら掃除機で吸うだけでもOKです。

頑固な油汚れはどうやって落とせばいいですか？

食器用中性洗剤を少し多めに溶かしたぬるま湯を使い、スポンジで数分なじませてからやさしくこすると効果的です。それでも落ちない場合は重曹水（水200mlに重曹小さじ1）を試してみてください。

網戸の掃除に塩素系漂白剤を使ってもいいですか？

基本的には不要です。また、塩素系漂白剤と酸性の洗剤を混ぜると有毒ガスが発生して非常に危険です。網戸の掃除には中性洗剤だけを使い、異なる種類の洗剤を絶対に混合しないでください。
