排水溝のぬめりの取り方【初心者でも簡単】

公開日: 2026-06-20

掃除排水溝ぬめりキッチンお風呂重曹クエン酸漂白剤初心者

排水溝のぬめりはなぜできるの？

排水溝のぬめりの正体は、主に「バイオフィルム」と呼ばれる細菌の集合体です。料理の油汚れ、石けんカス、皮脂、食べかすなどが排水溝に蓄積し、そこに雑菌が繁殖することでぬるぬるとしたぬめりが生まれます。放置するとにおいの原因になるだけでなく、排水の流れも悪くなってしまいます。でも大丈夫。正しい手順でしっかりと取り除くことができますよ。

掃除前に準備するもの

  • ゴム手袋（必ず着用してください）
  • 古い歯ブラシまたは排水溝専用ブラシ
  • 重曹（100円ショップやスーパーで購入可）
  • クエン酸または酢（食用・掃除用どちらでも可）
  • 塩素系漂白剤（カビキラー・キッチンハイターなど）※後述
  • バケツまたは洗面器
  • マスク（漂白剤使用時）

ゴム手袋は必ずつけましょう。ぬめりには雑菌が多く含まれており、素手で触れると手荒れや感染のリスクがあります。

ぬめりの取り方【ステップごとの手順】

ステップ1：パーツを取り外す

排水溝のフタ・ヘアキャッチャー・排水トラップなど、取り外せるパーツをすべて外します。外したパーツに付着したゴミや髪の毛は、ティッシュや古布でざっとふき取ってからゴミ箱に捨てましょう。ゴミを排水溝に流してしまうと詰まりの原因になるので注意してください。

ステップ2：重曹とクエン酸でぬめりを浮かせる

  1. 排水溝のまわりと取り外したパーツに重曹を適量（大さじ2〜3杯程度）ふりかけます。
  2. その上からクエン酸（大さじ1杯）を水50mlに溶かしたものをかけます。酢を使う場合は50〜100mlをそのままかけてもOKです。
  3. シュワシュワと泡立ったら、そのまま15〜30分放置します。
  4. 時間がたったら、古い歯ブラシでパーツをこすり洗いします。細かい溝や凹凸の部分も丁寧にこすりましょう。
  5. 最後にシャワーや水でしっかりと流します。

重曹のアルカリ性とクエン酸の酸性が反応して泡が発生し、汚れを浮かせてくれます。比較的マイルドな方法なので、週1回のルーティン掃除にも向いています。

ステップ3：塩素系漂白剤でしっかり除菌・カビ取り

重曹とクエン酸では落ちないしつこいぬめりやカビには、塩素系漂白剤（キッチンハイター・カビキラーなど）が効果的です。以下の手順で行いましょう。

  1. 窓を開けるなど換気をしっかり行います。マスクも着用してください。
  2. 取り外したパーツと排水溝内部に塩素系漂白剤を直接スプレーまたは塗布します。
  3. 5〜10分（製品の表示時間に従ってください）放置します。
  4. 水でしっかり洗い流します。

塩素系漂白剤は強力な除菌・漂白効果がありますが、使い方を誤ると危険です。次の注意点を必ず守ってください。

⚠️ 重要な安全注意事項

  • 塩素系漂白剤と酸性の洗剤（お酢・クエン酸・トイレ用洗剤など）を絶対に混ぜないでください。有毒な塩素ガスが発生し、大変危険です。重曹・クエン酸で掃除した後に塩素系漂白剤を使う場合は、水で十分に流してから使用してください。
  • 必ずゴム手袋とマスクを着用してください。
  • 使用中は必ず換気を行ってください。
  • 塩素系漂白剤は「混ぜるな危険」の表示を必ず確認しましょう。

ぬめりをきれいに保つためのコツ

  • 週1回、ざっと洗い流す習慣をつける：重曹をふりかけてお湯を流すだけでも予防効果があります。
  • 食べかす・油汚れをなるべく流さない：三角コーナーや排水溝ネットを活用し、固形物が流れないようにしましょう。
  • 使用後はなるべく水気を切る：湿った状態が長く続くと雑菌が繁殖しやすくなります。
  • 月1回は塩素系漂白剤で除菌：日常的には重曹＋クエン酸でケアし、月に一度は漂白剤でしっかりリセットするのがおすすめです。

よくある失敗と対処法

「こすっても全然落ちない」

放置時間が短い可能性があります。漂白剤や重曹をかけた後、規定の時間よりも少し長めに（ただし製品の上限を超えないように）置いてみましょう。また、歯ブラシよりも専用の排水溝ブラシを使うと、より深い部分までこすることができます。

「掃除してもすぐにぬめりが戻ってくる」

ぬめりの原因となる汚れが日常的に多く流れ込んでいる可能性があります。排水溝ネット（使い捨てタイプが便利）を取り付けて汚れをキャッチし、こまめに交換するようにしましょう。

「においが取れない」

排水トラップ（水がたまっているカップ状のパーツ）の奥側まで汚れが蓄積しているかもしれません。トラップも取り外して洗うか、パイプ専用の洗浄剤（パイプユニッシュなど）を使ってみましょう。

まとめ

排水溝のぬめりは、正しい道具と手順があれば初心者でも十分に取り除くことができます。日常的な予防（週1回の軽い掃除＋排水溝ネットの活用）と、月1回のしっかり除菌を組み合わせるのがベストです。一番大切なのは、塩素系漂白剤と酸性洗剤を絶対に混ぜないこと。安全に気をつけながら、清潔な排水溝をキープしましょう！
