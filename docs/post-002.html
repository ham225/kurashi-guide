電子レンジの掃除、後回しにしていませんか？

電子レンジは毎日使う家電ですが、内部の汚れを放置しているご家庭は意外と多いものです。食品の飛び散りや油汚れを放置すると、においの原因になるだけでなく、加熱効率が下がったり、こびりつきがひどくなって掃除がますます大変になったりします。

この記事では、特別な道具をほとんど使わずに、電子レンジをきれいにできる方法をステップごとに紹介します。読み終えたら、今日すぐに実践できますよ。

掃除を始める前に用意するもの

  • 耐熱容器（マグカップやボウルなど）
  • 水（約200ml）
  • キッチンペーパーまたは清潔な布巾
  • 中性洗剤（食器用で可）
  • 重曹（あれば）
  • ゴム手袋（肌が弱い方）

基本的にどのご家庭にもあるもので揃います。特別な洗剤を買い揃える必要はありません。

基本の掃除手順：水蒸気を使った方法

まず試してほしいのが「水蒸気を使った蒸らし掃除」です。水を温めてできた蒸気が汚れをふやかし、軽く拭くだけで落としやすくなります。

  1. 耐熱容器に水を入れる：マグカップや耐熱ボウルに水を約200ml入れます。重曹を小さじ1杯加えると、消臭効果が上がります。
  2. 電子レンジで加熱する：600Wで3〜5分加熱します。水が沸騰して内部に蒸気が充満すればOKです。
  3. そのまま5分待つ：加熱後すぐに扉を開けず、5分ほどそのままにして蒸気を汚れに浸透させます。
  4. 容器を取り出す：容器は大変熱くなっているので、必ずミトンやタオルを使って取り出してください。
  5. 内部を拭き取る：キッチンペーパーや布巾で天井・側面・底面をやさしく拭きます。ふやけた汚れがするりと落ちます。
  6. ターンテーブルを洗う：取り外せる場合は、ターンテーブルを食器と同じように中性洗剤で洗って乾燥させましょう。

頑固な油汚れには重曹ペーストを使う

水蒸気だけでは落ちない、こびりついた油汚れには重曹ペーストが効果的です。

  1. 重曹3：水1の割合で混ぜてペースト状にします。
  2. 汚れが気になる部分にペーストを塗り、5〜10分放置します。
  3. 湿らせたキッチンペーパーで優しく拭き取ります。
  4. 最後に水拭きして重曹が残らないようにします。

重曹は研磨作用があるため、強くこすりすぎると内部のコーティングを傷める可能性があります。やさしく拭き取るようにしてください。

外側・ドアの掃除もお忘れなく

内部だけでなく、電子レンジの外側やドアのゴムパッキン部分も汚れが溜まりやすい場所です。中性洗剤を薄めた水を布巾に含ませて拭き、その後乾拭きすると清潔に保てます。ゴムパッキンの溝には綿棒を使うと細かい部分まできれいにできます。

掃除をラクにするコツ

汚れたらすぐ拭く習慣をつける

加熱直後は庫内がまだ温かく、汚れが柔らかい状態です。食品が飛び散ったらすぐにキッチンペーパーで拭き取るだけで、こびりつきをほぼ防げます。

ラップや電子レンジ用カバーを使う

食品を温めるときにふんわりラップをかけるか、専用のシリコンカバーを使うと、飛び散りそのものを防げます。掃除の手間が大幅に減りますよ。

定期的なお手入れを習慣にする

週に1回、水蒸気を使った基本の掃除を取り入れるだけで、頑固な汚れがつきにくくなります。大掃除のときだけ慌てなくて済むようになります。

よくある失敗と注意点

塩素系漂白剤の使用は避けてください

「しっかり除菌したい」と塩素系漂白剤（カビキラーや塩素系の洗剤）を使う方がいますが、電子レンジ内部への使用はメーカーが推奨しておらず、臭いが残ったり内部を傷めたりする可能性があります。また、塩素系と酸性系の洗剤を混ぜると有毒ガスが発生する危険がありますので、絶対に混合しないでください。

金属製のたわしや硬いブラシは使わない

庫内の塗装やコーティングを傷つけてしまいます。必ず柔らかい布やキッチンペーパーを使いましょう。

加熱後の容器に注意する

水蒸気掃除のあと、容器は非常に熱くなっています。素手で触らず、必ずミトンやタオルを使って取り出してください。
