血液のシミの落とし方｜時間が経っても諦めない洗い方

公開日: 2026-06-30

洗濯シミ抜き血液掃除衣類ケア家事洗い方

血液のシミは「水」と「素早さ」がカギ

血液のシミは、付いてすぐなら水洗いだけでほぼ落とせます。しかし時間が経つと、血液中のタンパク質が繊維に固着して落ちにくくなります。このタンパク質が「曲者」で、お湯を使うと逆に固まってしまうため、必ず冷水（水道水）を使うのが大原則です。熱湯や温水は絶対に避けてください。

この記事では、衣類・シーツ・布製品についた血液のシミを、家にある道具で安全に落とす方法を手順ごとに説明します。

用意するもの

  • 冷水（ぬるま湯はNG）
  • 中性洗剤（食器用洗剤・おしゃれ着洗い洗剤など）
  • 歯ブラシまたはやわらかいブラシ
  • タオルまたは白い布（下に敷く用）
  • 重曹（頑固なシミ用・任意）
  • 酸素系漂白剤（時間が経ったシミ用・任意）

⚠️ 注意：塩素系漂白剤（カビキラーなど）と酸性の洗剤（クエン酸・酢など）を混ぜると有毒ガスが発生します。絶対に混合しないでください。

【ケース別】血液シミの落とし方

① シミが新鮮な場合（付いてすぐ〜数時間以内）

  1. シミ部分を冷水で裏側から流す。繊維の奥に押し込まないよう、裏から水を当てるのがポイントです。
  2. 中性洗剤を少量シミに直接つけ、指で優しくもみ込む。
  3. 冷水でしっかりすすいで、洗剤を流す。
  4. 落ちていなければ手順2〜3を2〜3回繰り返す。
  5. 通常通り洗濯機で洗う。

新鮮なシミであれば、これだけでほぼきれいになります。

② 時間が経ったシミ（乾いて固まっている場合）

  1. まず固まった血液を指や歯ブラシで優しくほぐし、表面の汚れを取り除く。
  2. シミ部分を冷水に30分〜1時間ほど浸け置きして柔らかくする。
  3. 中性洗剤を直接つけ、歯ブラシで繊維に沿って優しくこする（ゴシゴシこすると繊維が傷むので注意）。
  4. 冷水ですすいで落ち具合を確認する。
  5. まだ残る場合は、酸素系漂白剤（「ワイドハイター」など）を水に溶かし、30分〜2時間浸け置きする。
  6. 再度冷水ですすいでから洗濯機で洗う。

酸素系漂白剤は色柄物にも比較的安心して使えますが、必ず洗濯表示を確認してください。シルクやウール素材には使用できないものもあります。

③ カーペット・ソファなど洗濯できない布製品の場合

  1. ティッシュや乾いたタオルで血液を吸い取る（広げないよう外から内へ）。
  2. 冷水を含ませた布で、シミをそっと叩くように吸い取る。
  3. 中性洗剤を少量つけた布で同様に叩き洗いする。
  4. きれいな冷水を含ませた布で洗剤を拭き取る。
  5. タオルで水分をしっかり押さえて乾燥させる。

カーペットは濡れたまま放置するとカビが生えるため、仕上げにしっかり乾燥させることが重要です。

シミ落としをさらに効果的にするコツ

  • 重曹ペースト：重曹と冷水を2：1で混ぜたペーストをシミに塗り、15〜30分置いてから洗い流すと効果的です。
  • 食塩水：冷水に少量の塩を溶かした食塩水に浸け置きすると、タンパク質が分解されやすくなります。
  • すすぎ後に確認：乾燥させるとシミが目立たなくなったように見えますが、乾燥後は取れにくくなります。洗濯機から出したらすぐに確認しましょう。

よくある失敗と注意点

  • お湯を使う：タンパク質が熱で固まり、シミが永久に取れなくなることがあります。
  • シミを広げる方向にこする：外から内に向かって作業するのが基本です。内から外にこすると汚れが広がります。
  • 塩素系漂白剤の使用：色落ちや生地を傷める原因になります。血液シミには酸素系漂白剤を選びましょう。
  • 乾燥機に入れる：シミが残ったまま乾燥機にかけると、熱で完全に固着します。シミが落ちたことを確認してから乾燥させましょう。

まとめ

血液のシミ落としの基本は「冷水・素早く・優しく」の3つです。新鮮なうちなら中性洗剤と冷水だけで十分対応できます。時間が経ってしまった場合も、酸素系漂白剤を使った浸け置きで多くの場合きれいにできます。焦ってお湯を使ったり強くこすったりせず、手順通りに丁寧に対応してみてください。

よくある質問

血液のシミにお湯を使ってはいけないのはなぜですか？

血液に含まれるタンパク質は熱で固まる性質があります。お湯や熱湯を使うとタンパク質が繊維に強く固着し、シミが落ちにくくなるため、必ず冷水を使うことが重要です。

時間が経って乾いた血液のシミは落ちますか？

完全に落ちないこともありますが、冷水での浸け置きや酸素系漂白剤を使った方法で、多くの場合かなりきれいになります。シミが古いほど落ちにくいため、気づいたらできるだけ早く対処するのがおすすめです。

カーペットについた血液のシミはどうすればいいですか？

冷水を含ませた布でシミを外から内に向かって叩くように吸い取り、中性洗剤で同様に叩き洗いします。最後に水分をしっかり拭き取り、カビが生えないよう完全に乾燥させてください。

酸素系漂白剤と塩素系漂白剤、どちらを使えばいいですか？

血液のシミには酸素系漂白剤（ワイドハイターなど）を使いましょう。塩素系漂白剤は色落ちや生地へのダメージが大きく、白物以外には不向きです。また、両者を混合すると有毒ガスが発生するため絶対に混ぜないでください。
